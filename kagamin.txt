### ページ遷移
＜ヘッダー＞
    「ロゴ」　→　【ホーム画面】
    「検索バー」　→　【ホーム画面】　※検索結果を表示
    「アイコン」　→　メニュー出現
    「コーデを作る」　→　【コーデ作成画面（1. 服装選択画面）】

＜メニュー＞
    「マイページ」　→　【マイページ画面】
    「フォロー一覧」　→　【フォロー一覧画面】
    「フォロワー一覧」　→　【フォロワー一覧画面】
    「いいねしたファッション」　→　【いいね一覧画面】
    「設定」　→　【個人設定画面】
    「ログアウト」　→　【スタート画面】

【スタート画面】
    <Route path="/" element={<Start />} />
    ※最初の画面。「OutFit」のロゴを大きく目立たせたい。
    遷移：
        「新規登録」　→　【新規登録画面】
        「ログイン」　→　【ログイン画面】

【新規登録画面】
    <Route path="/sign-up" element={<SignUp />} />
    ※メールアドレス、パスワード、ユーザーネームを登録。
    遷移：
        「登録する」　→　【ホーム画面】　※成功したら

【ログイン画面】
    <Route path="/log-in" element={<Login />} />
    ※メールアドレスとパスワードが一致したらログイン。
    遷移： 
        「ログイン」　→　【ホーム画面】　※成功したら
        「パスワードが分からない」　→　【パスワード再設定用メール送信画面】

【パスワード再設定用メール送信画面】
    <Route path="/password-reset-email" element={<PasswordResetEmail />} />
    ※登録したメールアドレスを入力し、再設定用メールを送信する。
    ※メール送信ができればメッセージを表示する。遷移は設定しない。
    遷移：

【パスワード再設定画面】
    <Route path="/password-reset" element={<PasswordReset />} />
    ※新しいパスワードを設定する。
    遷移：
        「登録する」　→　【パスワード再設定成功画面】

【パスワード再設定成功画面】
    <Route path="/password-reset-success" element={<PasswordResetSuccess />} />
    ※新しいパスワードが設定されたことを表示し、ログイン画面への遷移を促す。
    遷移：
        「ログイン画面へ」　→　【ログイン画面】

【ホーム画面】
    <Route path="/home" element={<Home />} />
    ※全ての投稿を新着順で表示する。
    遷移：
        ＜ヘッダー＞
        「ファッション」or「タイトル」　→　【ファッション詳細画面】
        「作成者アイコン」「ユーザーネーム」　→　【ユーザー詳細画面】

【ファッション詳細画面】
    <Route path="/fashion-detail" element={<FashionDetail />} />
    ※該当の投稿の詳細を表示する。
    遷移：
        ＜ヘッダー＞
        「作成者アイコン」「ユーザーネーム」　→　【ユーザー詳細画面】

【ユーザー詳細画面】
    <Route path="/user-detail" element={<UserDetail />} />
    ※該当のユーザーの詳細を表示する。
    遷移：
        ＜ヘッダー＞
        「ファッション」or「タイトル」　→　【ファッション詳細画面】

【マイページ画面】
    <Route path="/my-page" element={<MyPage />} />
    遷移：
        ＜ヘッダー＞
        「マイページを編集する」　→　【マイページ編集画面】
        「ファッション」or「タイトル」　→　【ファッション詳細画面】
        「作成者アイコン」「ユーザーネーム」　→　【ユーザー詳細画面】

    遷移：
